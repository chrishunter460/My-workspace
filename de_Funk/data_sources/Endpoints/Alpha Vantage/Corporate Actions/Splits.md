## Description

Historical stock split events for a stock. Returns all splits with effective date and split ratio (e.g., 4:1 split means 4 new shares for every 1 old share).

Key for:
- Price adjustment calculations
- Share count normalization
- Historical return calculations
- Corporate action tracking

## API Response Format

```json
{
  "symbol": "AAPL",
  "data": [
    {
      "effective_date": "2020-08-31",
      "split_from": "1",
      "split_to": "4"
    },
    {
      "effective_date": "2014-06-09",
      "split_from": "1",
      "split_to": "7"
    }
  ]
}
```

## Request Notes

- Returns full split history
- Split ratio: `split_to / split_from` (e.g., 4/1 = 4.0 for a 4-for-1 split)
- Reverse splits have ratio < 1 (e.g., 1/10 = 0.1 for a 1-for-10 reverse split)
- All date fields in YYYY-MM-DD format

## Homelab Usage

```bash
python -m scripts.ingest.run_bronze_ingestion --endpoints splits --tickers AAPL TSLA NVDA GOOGL
```

## Known Quirks

1. **String numbers**: split_from and split_to returned as strings
2. **No reverse split flag**: Must infer from ratio < 1
3. **Historical depth**: Full history since IPO
4. **Timing**: Effective date is when split takes effect (market open)
