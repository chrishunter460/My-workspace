## Description

Historical and future (declared) dividend distributions for a stock. Returns all dividends paid including ex-dividend date, record date, payment date, and declaration date.

Key for:
- Dividend yield calculations
- Income portfolio analysis
- Ex-dividend date tracking for trading strategies
- Dividend growth analysis

## API Response Format

```json
{
  "symbol": "IBM",
  "data": [
    {
      "ex_dividend_date": "2024-02-08",
      "declaration_date": "2024-01-30",
      "record_date": "2024-02-09",
      "payment_date": "2024-03-09",
      "amount": "1.66"
    }
  ]
}
```

## Request Notes

- Returns full dividend history (20+ years for established companies)
- Future declared dividends included
- All date fields in YYYY-MM-DD format
- Amount returned as string, requires coercion to double

## Known Quirks

1. **String amounts**: Dividend amounts returned as strings
2. **Missing dates**: Declaration date may be null for older records
3. **Future dividends**: Declared but not yet paid dividends included
4. **Stock dividends**: Not included - only cash dividends
